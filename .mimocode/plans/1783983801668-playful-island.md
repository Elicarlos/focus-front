# Fix: Ranking + Login obrigatório

## Problem
- Ranking sem login = inauditável (qualquer um falsifica XP)
- Nickname prompt aparece toda vez

## Solution

### Regra: Ranking = Login obrigatório

| Estado | Comportamento |
|--------|---------------|
| Sem login | Mostra "Faça login para ver o ranking" com botão Google |
| Com login | Mostra ranking real com dados do backend |
| Apelido | Vem do Google (nome) — não precisa pedir |

### Mudanças

1. **`src/app/ranking/page.js`**
   - Se não tem token → mostra tela de login com botão Google
   - Se tem token → busca ranking do backend
   - Remover prompt de nickname (nome vem do Google)

2. **`src/app/dashboard/page.js`**
   - Quando loga com Google → salva nome do Google como nickname
   - Salva token em localStorage

3. **Backend (já existe)**
   - `/ranking` retorna top 50 por XP
   - Cada usuário tem nome do Google + XP real

### Fluxo
```
Usuário clica "Ranking"
  → Sem login? → "Faça login para ver o ranking" [Botão Google]
  → Com login? → Lista de ranking com dados reais
```

### Arquivos
- `src/app/ranking/page.js` — Requer login
- `src/app/dashboard/page.js` — Salva nome do Google no login

## Verificação
1. Sem login → ranking mostra tela de login
2. Login com Google → ranking mostra dados reais
3. Nome do Google aparece no ranking
4. XP sincronizado com backend
